# Smurfs

Easy way to build extensible CLI with Smurfs. Smurfs is helping to build MicroCLI Architecture

## Libraries

| Language       | Host | Client |
| -------------- | ---- | ------ |
| Go             | ✅ (Preview)    | ✅  (Preview)     |
| C#             | ⏳    | ⏳      |
| Node (TS & JS) | ❌    | ❌      |
| Java           | ❌    | ❌      |

## Go

Install Go library to host application

```bash

go get github.com/trendyol/smurfs/

```
